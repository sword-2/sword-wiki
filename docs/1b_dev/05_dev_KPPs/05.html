a2-6 Bible text assessments:

The scriptures are believed to be without error. For bible readers, some translations are more reliable than others which means some translations can receive recommendations for bible study and others just looked at from time to time. A table of recommendations is being derived (copied) from the (everyword) Sunday school class to be able to navigate a little faster. The table below is like a contents for class notes as they are input. the equivalence score is a number between 1 and 100. 1 represents a perfect formal equivalence (literal word-for-word) translation, and 100 perfect dynamic equivalence (thought-for-thought) translation.

If permissions are not available for recommended bibles, remaining translations, even if not recommended, may be practical at least for developing or maintaining software that would eventually load a recommended translation. This prevents a situation where a program can't be easily tested because no bible is available to load (no permissions). Also, the recommendations could be a priority for those doing programming or data entry if there is limited time, manpower, or finance available to keep up with every translation.
