a2-6 Bible text assessments:

Where there are limitations on a resource (time, manpower, etc) recommendations for English translations might help focus some efforts. One bible class by Brad Norman provided recommendations for some English translations as listed in the church references. These recommendations are being added to the table below as they are learned, Lord willing. The recommendations have the perspectives of how the translators did with formal and dynamic equivalence. Permissions and copyrights may not be a factor. There is an equivalence score which is a number between 1 and 100. 1 represents a perfect formal equivalence (literal), and 100 perfect dynamic equivalence. Comments
Recommendations
Translationrecommendationequivalence scorereading level
KJVnot recommended1512th gradeThe King James Version (KJV) was good for its time. But there is archaic language and new work by translators found there are some imperfections.
ASVnot recommended1212th gradeThe American Standard Version is better than KJV (better manuscript support, etc). But it is just as hard to read as the KJV.
RSVnot recommended2211th gradeThe Revised Standard Version (RSV) has theological problems and archaic pronouns.
NASB 2020recommenced1710th gradeOnly minor flaws capitalized pronouns, italics instead of brackets, all caps for old testament quotes. [For some reason one SWORD bishop app has an older NASB version, not the 2020]
